# OrangeBoy-keymap

QMK keymap for the Orange Boy Ergo keyboard by MechWild. Compile with QMK_MSYS, flash with qmk_toolbox.

Press and hold Boot0, tap NRST, release Boot0 after about 2 seconds to enter bootloader mode.

Macros:

Fn + a: ä

Fn + Shift + a: Ä

Fn + u: ü

Fn + Shift + u: Ü

Fn + o: ö

Fn + Shift + o: Ö

Fn + s: ß

Fn + e: €
